# IMAGINE: Cross-Domain MEG Decoding of Mental Imagery

## Overview

The **IMAGINE** project focuses on decoding human mental imagery from brain activity using Magnetoencephalography (MEG). The primary challenge is to analyze and classify MEG signals generated when subjects imagine one of 10 distinct object categories (apple, bicycle, brush, cake, clown, cup, desk, foot, mountain, zebra). 

The core of this problem lies in **cross-domain transfer**: learning spatial and temporal patterns from a "localizer" phase (where subjects are likely presented with the stimuli or explicit prompts) and applying these insights to an "imagery" phase (where subjects internally visualize or imagine the object). Because mental imagery signals are notoriously weak and highly variable across individuals, standard single-domain decoding techniques often fail.

## The Problem

1. **Low Signal-to-Noise Ratio (SNR):** MEG signals related to mental imagery are much weaker than those generated by direct sensory perception.
2. **Temporal Variability:** Different subjects exhibit the peak neural representation of their imagined objects at drastically different times. Some subjects are "early peakers" (e.g., showing strong signals at 1.3 seconds), while others are "late peakers" (e.g., maximizing around 2.7 to 3.4 seconds). A one-size-fits-all temporal window dilutes the signal with preceding or succeeding noise.
3. **Cross-Domain Shift:** The neural data distribution dynamically shifts between perception/localizer tasks and pure mental imagery.

## The Solution

To achieve robust decoding across a diverse set of subjects, this pipeline employs a heavily ensembled, temporally diverse classification approach. 

### 1. Preprocessing & Feature Extraction
- **MEG Modalities:** Uses both gradiometers (spatial gradients) and magnetometers (absolute magnetic fields) to capture complementary views of the neural activity.
- **Filtering:** Applies a 45Hz lowpass filter to remove high-frequency noise and prevent filter rolloff artifacts.
- **Frequency Bands:** Extracts band-power variance features across established neurophysiological bands (Theta: 4-8Hz, Alpha: 8-13Hz, Beta: 13-30Hz).

### 2. Multi-Window Temporal Strategy
Because subjects peak at different times, the pipeline extracts features over multiple overlapping and distinct imagery windows.
- **Wide Windows:** Captures sustained imagery signals (e.g., `[1.0s, 3.5s]`, `[0.5s, 4.0s]`).
- **Short Windows:** Targets specific temporal clusters (e.g., `[0.5s, 1.5s]` for early peakers, up to `[3.0s, 4.0s]` for late peakers) to maximize SNR for localized neural events.

### 3. Classification via Subject-Specific LDA
We rely on Linear Discriminant Analysis (LDA) applied on a per-subject basis, utilizing multiple strategies to handle the cross-domain shift:
- **Per-Domain Standardization:** Fits independent standard scalers for the localizer and imagery domains before classification, addressing the domain shift inherent between perception and imagery.
- **Shared-Norm Approaches:** Standardizes based on localizer data and applies it to very early imagery windows where distributions are closely aligned.
- **PSD (Power Spectral Density) Strategies:** Extracts Welch PSD features for continuous frequency representation, particularly leveraging gradiometer data.

### 4. Ensemble Voting
Instead of trying to find a single perfect feature, the system generates over 200 "votes" per trial. Each vote is an independent LDA prediction using a specific combination of:
- MEG channel type (mag or grad)
- Frequency band
- Localizer temporal window
- Imagery temporal window
- Standardization strategy

These predicted probabilities are averaged together to form a highly robust final ensemble prediction, ensuring that short bursts of high-SNR signal (captured by short windows) aren't completely washed out by the noise in broader temporal sweeps.
